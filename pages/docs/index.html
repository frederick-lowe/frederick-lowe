Atomic Author

  • The Control hierarchy reflects the following, common pattern: Controller > Handler > Service
  • Article Processing is performed in pipeline stages under $ROOT/src/Controllers/ronCrontroller.php
  • Some pipeline stages handle API calls. Others manage local steps, for example, image optimization
  • $ROOT/conf/src/Cron/[StageHandler].json defines "vendor"
  • "vendor" defines Service that will be invoked to handle a given stage. For example, if conf.src.Cron.FFHandler.vendor is set to "OpenAI", the Service used by the Handler will be $ROOT/src/Services/OpenAITextService.php
  • Service configuration is reposed in $ROOT/conf/src/Services/[ServiceName].json
